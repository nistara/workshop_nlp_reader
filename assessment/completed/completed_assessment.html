    # Loading a stopword list. Stop words are common words like "of," "the," etc.
    # We filter them out in text mining because they occur so much; if we didn't, 
    # these words would always be the predominating words in a document
    with open("data/voyant_stoplist.txt", 'r') as f:
        stopwords = f.read().split("\n")
    ​
    # We're appending a few extra words to the stopword list. This is common: it will 
    # depend on the research domain
    extra_words = ['dont', 'going', 'got', 'just', 'like', 'man', 'men', 'miss', 'mr', 'mrs', 'said', 'says', 'sir', 'things', 'way']
    stopwords = stopwords + extra_words
    ​
    # Changes all words in a document to their lowercase forms. This merges variants 
    # like "Table" and "table" so they can be counted together, rather than separately
    def to_lower(doc):
        return doc.lower()
    ​
    # Removes punctuation. In text mining, we are typically concerned with the words, 
    # not other characters. In fact, a computer would consider "calculate!" and
    # "calculate" to be two separate words (because of the !), so it's necesasry to 
    # remove extra characters. Students may mention that cleaning punctuation can 
    # be a multi-step process because of the differences in handling phrases like 
    # "self-taught" vs. "calculate!" (we'd replace the former with a space but the 
    # latter with nothing)
    def remove_punctuation(doc):
        doc = re.sub(r"[-]|[—]|[_]", " ", doc)
        doc = re.sub(r"[^\w\s]", "", doc)
        return doc
    ​
    # Removes digits, following the same logic for punctuation
    def remove_digits(doc):
        return re.sub(r"[0-9]", "", doc)
    ​
    # Removes extra whitespace, following the same logic for punctuation
    def remove_whitespace(doc):
        return re.sub(r"\s+", " ", doc)
    ​
    # This function actually does the work of removing the stop words loaded in 
    # at the beginning of this cell. Note too that it removes words less 3 characters 
    # long (students may do this)
    def remove_stop_words(doc):
        doc = doc.split()
        doc = [token for token in doc if token not in stopwords]
        doc = [token for token in doc if len(token) > 2]
        doc = ' '.join(doc)
        return doc
    ​
    # This is a 'helper' function, which controls all of the above. Instead of calling 
    # each function individually when loading the files, we use `clean()` to do it 
    # for us
    def clean(doc):
        lowercase = to_lower(doc)
        no_punct = remove_punctuation(lowercase)
        no_digits = remove_digits(no_punct)
        no_whitespace = remove_whitespace(no_digits)
        stopped = remove_stop_words(no_whitespace)
        return stopped

    **4b.** Students are required to fine-tune their model. In addition to changing the number of topics, this will involve setting two hyperparameters, alpha and eta. The higher the alpha, the more evenly distributed topic proportions are in a particular document; a lower alpha means topic proportions are less evenly distributed. The higher the eta, the more word probabilities will be distributed evenly across a topic (specifically, this boosts the presence of low-probability words); a lower eta means word distributions are more uneven, so each topic will have less dominant words.
    ​
